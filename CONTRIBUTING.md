# Contribution Guidelines

Thank you for considering contributing to greensky-online! We value community involvement and appreciate your time and effort.

## How to Contribute

1. Fork the repository.
2. Clone your fork to your local machine.
3. Create a new branch for your feature or fix.
4. Make your changes following the coding standards.
5. Commit your changes with clear messages.
6. Push the branch to your fork.
7. Open a Pull Request to the main branch.

## Reporting Issues

If you find a bug or unexpected behavior:
- Use the GitHub Issues tab.
- Provide clear steps to reproduce the issue.
- Include screenshots or logs if applicable.

## Suggesting Features

We welcome feature suggestions!
- Open an issue with the label **feature request**.
- Clearly describe the idea and its benefits.

## Code of Conduct

Be respectful, inclusive, and supportive.
Harassment, discrimination, or offensive behavior will not be tolerated.

## Review Process

- All submissions are reviewed by maintainers.
- Feedback may be provided for improvements.
- Once approved, changes will be merged into the main branch.
